# Intensifier

Intensifier intensifies.

## Usage

```
./intensifier -h

  -expire duration
        time to expire images after (default 336h0m0s)
  -font string
        font to use (default "/usr/share/fonts/truetype/msttcorefonts/impact.ttf")
  -port int
        port to listen on (default 8080)
```
